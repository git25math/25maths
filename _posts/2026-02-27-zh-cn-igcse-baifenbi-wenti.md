百分比题出现在每一张 IGCSE 数学试卷中。从1分的简单计算到5分的复利题。以下是每种题型和考官期望的精确方法。

## 题型1：求一个数的百分比

**方法：** 数量 × (百分比 / 100)

**例：** 求 $240 的 15%。
```
240 × 15/100 = 240 × 0.15 = $36
```

最简单的题型。常见错误是忘记除以 100。

## 题型2：百分比增减

**增加：** 新值 = 原值 × (1 + 百分比/100)
**减少：** 新值 = 原值 × (1 - 百分比/100)

**例：** 一件衬衫 $80，打7折。求售价。
```
80 × (1 − 30/100) = 80 × 0.70 = $56
```

**乘数捷径：**

| 变化 | 乘数 |
|------|------|
| 增加 5% | × 1.05 |
| 增加 12% | × 1.12 |
| 减少 20% | × 0.80 |
| 减少 35% | × 0.65 |

## 题型3：一个量占另一个量的百分比

**方法：** (部分 / 整体) × 100

**例：** 180 名学生中 45 人选了数学。百分比是多少？
```
(45 / 180) × 100 = 25%
```

## 题型4：百分比变化

**方法：** ((新值 − 原值) / 原值) × 100

**关键：** 分母始终是**原值**，不是新值。

## 题型5：逆向百分比（求原值）

**这是大多数学生最头疼的题型。** 给你变化后的最终金额，要求原值。

**方法：** 最终金额 / (1 ± 百分比/100)

**例：** 涨价20%后，电视售价 $360。求原价。
```
360 / 1.20 = $300
```

**致命错误：** 学生计算 $360 的 20% 再减去。这给出 $288，是错的。20% 是对原值而不是 $360 计算的。

## 题型6：复利

**公式：** A = P(1 + r/100)^n

- A = 最终金额
- P = 本金
- r = 年利率
- n = 年数

**例：** $5000 以年利率 3% 复利投资 4 年。
```
A = 5000 × 1.03^4 = $5627.54
```

## 题型7：折旧

同复利但用减法：**V = P(1 − r/100)^n**

## 题型8：求利率或时间

已知 A = P(1 + r/100)^n，可能需要求 r 或 n。

**求 n：** 用试值法。

**例：** $1000 以 5% 复利增长到 $1200。几年？
```
n=3: 1.05^3 = 1.158 (不够)
n=4: 1.05^4 = 1.216 (超过 1.2)
答案: 需要 4 个完整年
```
