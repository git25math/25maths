If you're choosing between CIE 0580 and Edexcel 4MA1, or your school offers both, this comparison covers every practical difference that affects your exam preparation.

## Exam Structure

| Feature | CIE 0580 (Extended) | Edexcel 4MA1 (Higher) |
|---------|---------------------|----------------------|
| Number of papers | $2$ (Paper 2 + Paper 4) | $2$ (Paper 1H + Paper 2H) |
| Total time | $4$ hours | $3$ hours |
| Total marks | $200$ ($70 + 130$) | $200$ ($100 + 100$) |
| Calculator | Paper 2: Yes, Paper 4: Yes | Paper 1H: No, Paper 2H: Yes |
| Grade range | A* to E | $9$ to $4$ |

**Key difference:** CIE has no non-calculator paper. Edexcel Paper 1H is non-calculator, which tests mental maths and algebraic manipulation more heavily.

## Topic Coverage

Both syllabuses cover the same core areas, but the depth varies:

| Topic | CIE 0580 | Edexcel 4MA1 | Difference |
|-------|----------|-------------|------------|
| Algebra | Up to quadratic formula, completing the square | Similar depth | Roughly equal |
| Number | Standard form, bounds, ratio | Same | Roughly equal |
| Geometry | Circle theorems, similarity | Circle theorems, congruence proofs | Edexcel has more proof |
| Trigonometry | Sine/cosine rules, 3D trig | Same | Roughly equal |
| Statistics | Cumulative frequency, histograms | Same + box plots | Edexcel adds box plots |
| Probability | Tree diagrams, conditional | Tree diagrams, Venn diagrams | CIE emphasises Venn diagrams more |
| Functions | Domain, range, composite, inverse | Same | Roughly equal |
| Vectors | 2D vectors, proofs | Same | Roughly equal |
| Calculus | Not included | Not included | Neither |
| Sets | Full set notation, Venn diagrams | Limited | CIE has more set theory |
| Matrices | Included (Extended) | Not included | CIE only |
| Transformations | All $4$ types + matrices | All $4$ types | CIE adds matrix transformations |

## Which Is "Harder"?

Neither is universally harder. The difficulty depends on your strengths:

**CIE 0580 is harder if you:**
- Struggle with time pressure (Paper 4 is $2$ h $30$ min for $130$ marks)
- Find matrices and set theory difficult
- Need more structured questions (CIE questions are often less scaffolded)

**Edexcel 4MA1 is harder if you:**
- Rely heavily on a calculator (Paper 1H has none)
- Find geometric proofs challenging
- Struggle with non-routine problem-solving (Edexcel problem-solving questions can be creative)

## Grade Equivalence

| CIE 0580 | Edexcel 4MA1 | UK Equivalent |
|----------|-------------|---------------|
| A* | $9$/$8$ | A* |
| A | $7$ | A |
| B | $6$ | B |
| C | $5$/$4$ | C |

Universities generally treat both as equivalent. Neither gives an advantage over the other for applications.

## Exam Timing

| Session | CIE 0580 | Edexcel 4MA1 |
|---------|----------|-------------|
| May/June | Yes | Yes |
| October/November | Yes | January (Paper 1H/2H) |
| Resit options | Next session | Next session |

CIE offers $2$ full sessions per year. Edexcel has a January session for some regions.

## Preparation Strategy Differences

**For CIE 0580:**
- Practice matrices and set theory (unique to CIE)
- Focus on long-answer technique (Paper 4 has $8$–$10$ mark questions)
- Build speed — every question requires a calculator, so efficiency matters

**For Edexcel 4MA1:**
- Practise mental arithmetic and estimation (Paper 1H)
- Learn geometric proof language ("hence show that...")
- Master non-calculator algebra manipulation

## Switching Between Boards

If you've prepared for one board and need to switch:
- **CIE → Edexcel:** Learn non-calculator techniques, drop matrices
- **Edexcel → CIE:** Learn matrices, set theory, practice longer papers

The overlap is roughly $85\%$. Most of your preparation transfers.
