Paper 4（Extended）是 CIE 0580 中最长、权重最高的试卷。2小时30分钟完成130分，节奏和技巧与知识同样重要。

以下是帮助学生多拿20+分的10个具体技巧。

## 1) 按分值分配时间，而不是按题目

**规则：** 1分 ≈ 1分10秒。

6分的题值7分钟。1分的题只给70秒。
在一道难题上花15分钟的学生会在其他地方丢掉容易的分数。

## 2) 从你最擅长的部分开始

Paper 4 不要求按顺序作答。先花2分钟浏览所有题目，然后从你最熟悉的主题开始。早期的信心会改善后面难题的表现。

## 3) 代入前先写公式

对于任何基于公式的题目（测量、三角函数、复利）：

```
第1步：写出公式
第2步：代入数值
第3步：计算
```

即使最终答案错误，这也能获得过程分。

## 4) 用"反代验证"检查代数

解完方程后，**把答案代回原方程**。
这只需30秒，却能发现每道题可能丢2-3分的符号错误。

## 5) 即使不要求也画图

涉及以下内容的题目：
- 方位角 → 画北方向线
- 三角函数 → 画出标注边长的三角形
- 概率 → 画树状图或样本空间

图形减少错误，有时还能得到过程分。

## 6) "证明"题是白送的分

当题目说"证明 x = 7.5"时，答案已经给你了。你的任务是展示清晰的过程。

**常见错误：** 学生因为知道答案就跳步骤。考官需要看到每一行。

## 7) 掌握3种图表题类型

| 类型 | 做什么 |
|------|--------|
| 画图 | 用尖铅笔仔细标点，用光滑曲线连接（不是直线） |
| 读图 | 用尺子画辅助线，按要求的精度读取 |
| 描述变换 | 说明所有必要细节（如反射：反射线；放大：比例因子和中心） |

## 8) 统计题：始终检查 n 值

在累积频率和直方图题目中，最常见的错误是使用错误的总数。

- 从题目或表格中读取"n"
- 检查你的累积频率终值是否等于 n
- 中位数：用 n/2（分组数据不用 (n+1)/2）

## 9) 向量题：标注一切

向量题经常是多步串联。清楚标注每个向量：

```
→    →    →
AB = AO + OB = -a + b = b - a
```

搞混方向（A→B vs B→A）是向量题的头号错误。

## 10) 最后10分钟用来检查

**不要**在最后10分钟开始做新题，除非你真的已经做完了。

用这段时间：
1. 重读你觉得简单的题目（审题失误丢分最多）
2. 检查所有答案是否有正确的单位
3. 确认"精确答案"题没有小数
4. 数一数你是否答了每一小题 (a)、(b)、(c)
