# Mastery Series Week 1 Build

This folder contains the minimum executable build for the first 4-week phase.

## Deliverables

1. Unified LaTeX visual system (`_templates/latex`)
2. First complete product skeleton: `quadratics-mastery-pack`
3. Reusable bilingual sales page template (`_templates/sales`)

## Structure

- `_templates/latex`: reusable visual design system for all future packs
- `_templates/sales`: reusable listing page structure for Payhip
- `quadratics-mastery-pack`: first sellable pack scaffold with 6 modules

## Build Notes

- Recommended engine: `pdflatex` (also compatible with `xelatex`)
- Keep module files internal and package externally as one public product bundle.
- Use this directory as the source of truth for Week 1 assets.
