# Module 4 - Vocabulary Sheet (EN + ZH)

## Purpose

Make mathematical language explicit so students can explain methods, justify choices, and avoid avoidable communication errors in exam answers.

## Core Terms (Exam-Ready)

| Term (EN) | 中文 | Precise Exam Meaning | Typical Misuse | Quick Fix Prompt |
|---|---|---|---|---|
| quadratic equation | 二次方程 | Equation where highest power of variable is 2 | Calls any expression with x^2 an equation | "Is there an equals sign?" |
| quadratic expression | 二次代数式 | Expression of form ax^2+bx+c without equation sign | Treated as solvable directly | "Set expression equal to what?" |
| root / solution | 根 / 解 | Value of x making equation true | Writes only one root | "How many roots are expected?" |
| factorise | 因式分解 | Rewrite as product of factors | Mixes with expansion | "Can it be written as ( )( ) ?" |
| expand | 展开 | Multiply brackets to standard form | Used when asked to factorise | "Direction: compact or expanded?" |
| discriminant | 判别式 | b^2-4ac, predicts root nature | Wrong b sign in calculation | "Substitute b with brackets" |
| completing the square | 配方法 | Rewrite to (x+p)^2+q form | Forgets balancing constant | "What was added and subtracted?" |
| axis of symmetry | 对称轴 | Vertical line through vertex x-value | Confused with y-intercept | "Is this a line x=... ?" |
| vertex | 顶点 | Max/min turning point of parabola | Sign read incorrectly from form | "Check with y=(x-h)^2+k" |
| x-intercept | x 轴截距 | Point where graph crosses x-axis, y=0 | Reported as y-value only | "Write as coordinate" |
| y-intercept | y 轴截距 | Point where graph crosses y-axis, x=0 | Mixed with root | "Substitute x=0" |
| repeated root | 重根 | Same root occurs twice, discriminant 0 | Treated as two different roots | "Does equation look like (x-a)^2?" |
| real roots | 实根 | Roots on real number line | Includes complex roots by mistake | "Check discriminant sign" |
| no real roots | 无实根 | Discriminant < 0 in real-number context | Attempts invalid square root as real | "Stop at classification" |
| interval notation | 区间表示 | Compact inequality solution form | Bracket direction errors | "Boundary included or excluded?" |

## Method-Language Frames

### Method Choice
- EN: `I selected [method] because [structure cue].`
- ZH: `我选择 [方法]，因为题目结构出现了 [特征]。`

### Multi-Root Final Line
- EN: `Therefore, the roots are x = ..., x = ... .`
- ZH: `因此，方程的两个根为 x = ...，x = ...。`

### Context Filter
- EN: `From the two algebraic roots, only x = ... is valid in context because ... .`
- ZH: `在两个代数解中，只有 x = ... 符合题意，因为......。`

### Discriminant Conclusion
- EN: `Since b^2-4ac = ... < 0, there are no real roots.`
- ZH: `因为 b^2-4ac = ... < 0，所以该方程没有实根。`

## Quick Oral Drill (3 Minutes)

1. Teacher states equation type; student responds with best method term.
2. Teacher shows solution; student states full final-answer sentence.
3. Teacher gives word-problem root pair; student rejects invalid root verbally.
