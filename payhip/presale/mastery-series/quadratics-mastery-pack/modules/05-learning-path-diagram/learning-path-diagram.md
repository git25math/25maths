# Module 5 - Learning Path Diagram Notes

## Purpose

Translate the five-stage mastery model into an execution path that can be used by students, parents, and teachers.

## Path Rules

1. Stage 1 (`Diagnose`) is mandatory before any speed practice.
2. Stage 2 (`Build`) must include both method practice and language precision.
3. Stage 3 (`Apply`) should mix pure algebra and context transfer questions.
4. Stage 4 (`Correct`) targets top 2 recurring error tags only.
5. Stage 5 (`Reinforce`) uses spaced Kahoot sets before final readiness check.

## Weekly Rhythm (Minimal)

- Day 1: Stage 1 + Stage 2
- Day 2: Stage 2 + Stage 3
- Day 3: Stage 3 + Stage 4
- Day 4: Stage 4 + Stage 5
- Day 5: mixed retest and progress logging

## Gate Criteria

Advance only if:
- Missing-root errors are zero in latest mixed set.
- Context-root validation is explicit in all word problems.
- Method choice is stated for at least 80% of questions.
