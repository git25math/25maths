# Module 3 - Error Pattern Guide

## Purpose

Convert repeated quadratic mistakes into explicit correction loops tied to workbook question IDs.

## Error Tag Matrix (Aligned to Q01-Q24)

| Error Tag | Symptom | Root Cause | Primary Linked QIDs | Fast Fix Drill |
|---|---|---|---|---|
| `QP-01-missing-root` | One valid root is omitted on final line | Final-answer format habit not automatic | Q01, Q02, Q03, Q07, Q19 | Write both roots in one final sentence for 5 solved equations |
| `QP-02-sign-slip` | Rearrangement or factor signs are wrong | Sign tracking is not line-checked | Q04, Q06, Q08 | Add a sign-check line before factorising |
| `QP-03-method-mismatch` | Slow or unsuitable method selected | No method-selection rule | Q09, Q10, Q13, Q15, Q20, Q23 | Use a 3-choice method card before solving |
| `QP-04-invalid-context-root` | Algebraic root kept without context filter | Domain check missing | Q17, Q18, Q24 | Add a context-validity sentence after every model |
| `QP-05-arithmetic-load` | Formula setup correct but arithmetic fails | Calculation load too high in one line | Q05, Q11, Q12, Q14 | Split substitution and simplification into separate lines |
| `QP-06-discriminant-misread` | Root nature or graph inferences are incorrect | Weak discriminant interpretation | Q16, Q21, Q22 | Run `Delta > 0 / =0 / <0` mini-table before conclusion |

## Intervention Flow

1. Mark workbook and collect top 2 tags for each learner.
2. Assign one micro-drill per tag (6-10 minutes each).
3. Re-run a mixed set using at least 2 questions from each top tag.
4. Compare tag frequency after two cycles; keep logs for progression.

## Drill Bank (Minimum)

- `DR-01-root-format`: 6 equations, output line must include both roots.
- `DR-02-sign-control`: 8 rearrangement lines, check each sign transform.
- `DR-03-method-choice`: 12 equations, pick method before solving.
- `DR-04-context-filter`: 6 word problems, reject invalid roots explicitly.
- `DR-05-formula-arithmetic`: 6 formula questions with line-separated arithmetic.
- `DR-06-discriminant-logic`: 10 equations, classify root type first.
