1 How To Use This Workbook

• Complete blocks in order: A → B → C.
• Write method selection before algebra steps.
• Keep final answers in exam format (all roots stated).
• Use Module 2 to mark and Module 3 to diagnose repeated errors.

  Key Concept
  Quadratics performance improves fastest when students train method selection first, then execution
  speed.

  Exam Tip
  Before writing the final line, run a two-step check: (1) all roots listed, (2) root values satisfy the
  original equation.
