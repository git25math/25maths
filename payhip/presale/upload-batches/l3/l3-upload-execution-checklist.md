# L3 Upload Execution Checklist

Use this checklist while creating/updating Payhip listings for `L3`.

## Batch Summary

- Level: `L3`
- SKU count: `15`
- Input batch CSV: `payhip/presale/upload-batches/payhip-upload-batch-l3.csv`
- Ops sheet: `payhip/presale/upload-batches/l3/l3-ops-sheet.csv`
- URL backfill template: `payhip/presale/upload-batches/l3/l3-url-backfill-template.csv`

## Per-SKU Required Steps

1. Create/update Payhip product with `final_title_en`.
2. Set price (`price_early_bird` + `price_regular`).
3. Paste description + subtitle + SEO fields.
4. Upload 9 images (`main_cover_png` first, then 8 supporting images).
5. Publish product and paste `final_payhip_url` in ops sheet.
6. Mark all status columns for that SKU.
