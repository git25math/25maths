This document used to describe the website while the legacy online exercise product line was still active. That product line has since been removed from the website, so the original 2026-04-03 analysis is no longer a reliable implementation source.

Use these current sources instead:

- `docs/DEVELOPMENT-PLAN.md` for the active website scope and retired-surface guardrails.
- `docs/CONTRIBUTING.md` for paths and schema that must not be restored.
- `supabase/README.md` for the current database verification model.
- `plan/MEMBER-SYSTEM-COMMAND-CENTER.md` for member-system scope.

Current active product surfaces are worksheet/resource pages, Payhip-backed downloads, Kahoot packs, member entitlement/download flows, and institution resource planning. Any future practice or exam-board question system should be designed as a new product surface with its own data model, not by reviving the retired website exercise implementation.
