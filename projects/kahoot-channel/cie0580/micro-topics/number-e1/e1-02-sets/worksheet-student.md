## Syllabus focus
- Use set notation, complements, union, intersection, and simple Venn relationships.

## Model example
Given `U = {1,2,3,4,5,6}`, `A = {2,4,6}`, `B = {3,6}`:
- `A intersection B = {6}`
- `A union B = {2,3,4,6}`

## Practice (10)
1. Write the set of factors of `12`.
2. If `A = {1,2,3}` and `B = {3,4}`, find `A intersection B`.
3. If `A = {1,2,3}` and `B = {3,4}`, find `A union B`.

8. If `A` is a subset of `B`, explain what this means.
9. In a Venn diagram, what does the overlap represent?
10. In `50` students, `28` like maths, `20` like physics, and `9` like both. How many like neither?
