## Syllabus focus
- Interpret rounded values using upper and lower bounds, and use intervals in calculations.

## Model example
If `x = 5.6` correct to 1 d.p., then `5.55 <= x < 5.65`.

## Practice (10)
1. State interval for `x=9` to nearest integer.
2. State lower and upper bounds for `y=4.3` to 1 d.p.
3. Length 18 cm to nearest cm: interval for `L`.
4. Mass 2.50 kg to nearest 0.01 kg: interval.
5. If `a in [3.5,4.5)` and `b in [1.5,2.5)`, find min `a+b`.
6. Same intervals, find max `a+b`.
7. Rectangle length 10 cm nearest cm, width 6 cm nearest cm. Find max area.
8. Same rectangle, find min area.
9. Distance 120 km nearest km, time 2 h nearest 0.1 h. Which bounds give max speed?
10. Write interval for `p=0.72` correct to 2 d.p.
