# Launch Pipeline Checklist (Per Micro-Topic)

## Required Assets
- [ ] Kahoot title + description + tags
- [ ] Kahoot MCQ CSV/markdown question set
- [ ] Cover PNG (580x380) + source SVG
- [ ] Worksheet student PDF
- [ ] Worksheet answers PDF (next page)
- [ ] Payhip short copy
- [ ] Payhip long copy
- [ ] Price tier assigned

## Upload Steps
1. Upload Kahoot and publish to correct channel.
2. Upload product files to Payhip (PDF + bonus source if needed).
3. Add product image (same visual family as Kahoot cover).
4. Set price and category (`CIE0580` or `Edexcel4MA1`).
5. Add CTA links both ways:
- Kahoot description -> Payhip
- Payhip description -> Kahoot channel
6. Add email tagging field in lead forms:
- `board`
- `topic`
- `micro_topic`

## QA Before Publish
- [ ] Thumbnail readable at small size
- [ ] No black border / no dark frame
- [ ] Correct answer keys verified
- [ ] Math notation consistent
- [ ] Links open correctly
