# Kahoot Closed-Loop Production Plan (12 Topics)

## Goal
Build a conversion loop from Kahoot audience to email list to paid products:
- `Kahoot` -> `Topic worksheet lead magnet / paid worksheet` -> `Email segmentation` -> `Payhip single pack / bundle / subscription`.

## Output Unit (Definition of Done)
Each micro-topic must include:
1. Kahoot question set (12-20 MCQ, exam-style distractors)
2. Cover (580x380, warm minimal style)
3. Listing copy (title, description, tags)
4. Worksheet student page (syllabus note + model + 10 questions)
5. Worksheet answer page (next page, mark-scheme style)
6. Payhip listing copy (value stack + CTA + FAQ)
7. Price and offer flag (free/paid/bundle)

## Packaging Strategy (Recommended)
- Per macro-topic: release 6 micro-topics.
- Free: first 2 micro-topics (lead magnet).
- Paid single: next 4 micro-topics.
- Bundle: all 6 micro-topics with discount.

## Pricing Strategy (USD)
- Free starter pack (2 micro-topics): `$0` (email required)
- Single micro-topic pack (Kahoot + worksheet + answers): `$3.90`
- Topic bundle (6 micro-topics): `$14.90`
- Board bundle launch (12 macro-topic highlights): `$39.00`
- Optional rolling membership (new weekly packs): `$12/month`

Price logic:
- Single pack low-friction entry.
- Bundle target 3.8x single value perception.
- Membership only after weekly release cadence is stable (>= 8 weeks).

## 8-Week Execution Sprint

### Week 1-2: Build and Standardize
- Freeze palette + cover template
- Finish CIE topics 01-03 full deliverables
- Build Payhip product template and tagging taxonomy

### Week 3-4: First Revenue Loop
- Complete CIE topics 04-06
- Launch first 3 free lead magnets + 3 paid packs
- Start email segmentation by topic interest

### Week 5-6: Board Expansion
- Replicate to Edexcel 4MA1 topics 01-06
- Push cross-board upsell in email sequence

### Week 7-8: Bundle and Optimization
- Complete 12/12 topic packs
- Launch board bundle and limited-time coupon
- A/B test listing headline + first image

## Content QA Checklist
- Question wording is exam-appropriate and unambiguous
- Wrong options reflect real misconceptions
- Worksheet difficulty progression: easy -> medium -> exam-style
- Answer page includes method cues for partial credit
- Cover text readable in small thumbnail
- Payhip listing includes audience, outcomes, and usage rights

## Operational Tracking Fields
Track every micro-topic with:
- `board`
- `macro_topic`
- `micro_topic_code`
- `kahoot_url`
- `lead_or_paid`
- `payhip_url`
- `email_tag`
- `status` (`draft`/`uploaded`/`live`)
