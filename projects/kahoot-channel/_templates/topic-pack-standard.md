# Topic Pack Standard (Mandatory)

Use this standard for every micro-topic (small syllabus point).

## 1. Scope Rule
- One pack = one micro-topic (single assessable syllabus point).
- Do not mix multiple skill families in one pack.
- Every pack must be exam-board specific (CIE 0580 or Edexcel 4MA1).

## 2. Required Outputs Per Micro-Topic
1. `kahoot-question-set.md`
- Syllabus point mapping (code/statement)
- Past-paper question type mapping
- 15-20 MCQ with correct option + distractor logic
- Upload title/description/tags

2. `worksheet-pack.tex`
- Page 1: worksheet (student version)
- Page 2: answer key + concise worked solutions

3. `listing-copy.md`
- Product title
- One-line hook
- Short description
- Includes list
- CTA

4. Cover assets
- `cover-2320x1520-kahoot-minimal.svg`
- `cover-2320x1520-kahoot-minimal.png`
- Must use track badge (`Core Track` or `Extended Track`) matching syllabus tier.

## 3. Alignment Checks (Must Pass)
- Syllabus statement and questions are directly aligned.
- Question styles mirror common exam wording and mark patterns.
- Worksheet questions map to Kahoot objectives.
- Answer page includes final answer and method note.

## 4. File Naming
- Topic folder: `NN-micro-topic-slug`
- Required files:
  - `kahoot-question-set.md`
  - `worksheet-pack.tex`
  - `listing-copy.md`

## 5. Publish Gate
- [ ] Math accuracy checked
- [ ] Distractors checked
- [ ] Worksheet/answers page split verified
- [ ] PDF exported successfully
- [ ] Listing copy finalized
