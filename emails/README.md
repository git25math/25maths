# Email Templates (Waitlist + Launch)

These templates support the free → trust → paid conversion flow.
Replace placeholders before sending.

Placeholders:
- {{name}}
- {{link}} (free resource download)
- {{payhip_link}} (paid product link)
- {{early_bird_offer}} (optional)

Suggested early‑bird offers (pick one):
- 15% off for waitlist subscribers
- $3 off any bundle on launch week
- Free bonus worksheet with purchase

Membership templates:
- `06-membership-welcome.md`
- `07-membership-weekly-update.md`
- `08-membership-renewal.md`

Additional placeholders for membership:
- {{member_hub_link}}
- {{weekly_pack_link}}
- {{answer_link}}
- {{solutions_link}}
- {{period_end}}
- {{renew_link}}
