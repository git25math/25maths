# Interactive Exercises Task Tree

## Objective

将本对话中关于 `interactive exercises` 的目标、决策、执行记录与验证证据，整体迁移到独立任务树，确保后续迭代与其他并行流（如 member、payhip、bilingual）隔离执行，避免相互冲突。

## Rules Of Execution

1. 仅处理 interactive exercises 相关范围：入口、筛选、题库一致性、练习页联动、健康检查与发布验证。
2. 每次执行必须写入 `EXECUTION-LOG.md`。
3. 关键策略决策先登记到 `DECISIONS-LEDGER.md`。
4. 发现异常或回归时，先记入 `ISSUES-AND-FIXES.md`，再执行修复。
5. 任何新增范围先写入 `CHANGE-CONTROL.md`。
6. 不修改其他任务树所有权文件，除非有明确授权。
